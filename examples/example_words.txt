# 单词表示例文件
# 
# 语法说明:
# - 普通单词: 直接写单词
# - 圆括号 (): 强制指定读音
# - 方括号 []: 强制指定查词典形态
# - 组合使用: 单词(读音)[查词形态]

# ===== 基本格式 =====
精霊
炎
柱

# ===== 指定读音 (当词典有多个读音时) =====
精霊(せいれい)
生贄(いけにえ)
煉獄(れんごく)

# ===== 指定查词形态 (活用形/复合词) =====
食べた[食べる]
狂い咲く[狂い咲き]
走った[走る]

# ===== 组合使用 =====
走った(はしった)[走る]
食べてる(たべてる)[食べる]

# ===== 片假名词汇 (自动处理) =====
ピン
コーヒー
アニメ

# ===== 复合词示例 =====
立ち向かう
振り返る
見つめる

# ===== 常见活用形 =====
見た[見る]
来た[来る]
した[する]
言った[言う]
思った[思う]
